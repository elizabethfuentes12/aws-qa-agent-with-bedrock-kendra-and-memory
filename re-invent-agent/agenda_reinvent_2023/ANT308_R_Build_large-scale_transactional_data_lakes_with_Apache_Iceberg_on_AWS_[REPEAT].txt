description: <p>Apache Iceberg is an open table format for very large analytic datasets. Apache Iceberg manages large collections of files as tables, and it supports modern analytical data lake operations such as record-level insert, update, delete, and time travel queries. The Iceberg specification allows for seamless table evolution and its design is optimized for usage on Amazon S3. In this builders’ session, learn how to build a transactional data lake with Apache Iceberg and extend it for cataloging, analytics, and data processing with AWS analytics services such as AWS Glue, Amazon Athena, Amazon EMR, and Amazon Redshift.
