description: <p>At Netflix, teams periodically need to derive AWS resource utilization insights and attribute them for consumption by a heterogeneous set of stakeholders. A frequent problem encountered during this task was the ability to meaningfully decompose and resynthesize slices of fundamental AWS resources into Netflix resources (created or provisioned by various Netflix services). This chalk talk covers how Netflix applied their exploratory work, learnings from various interviews, and consultation with AWS technical account managers to build an ecosystem to help track ownership and cost attribution metadata for multi-tenant Amazon S3 buckets using Netflix’s metadata and Amazon S3 Storage Lens.</p>
